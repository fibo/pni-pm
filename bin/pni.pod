
=head1 NAME

pni

=head1 SYMOPSIS

    $ pni --help

=head1 DESCRIPTION

It is a L<Mojo::Command> pointing to the "PNI::Command" namespace.

=head1 COMMANDS

=head2 daemon

    $ pni daemon

=head1 ENVIRONMENT VARIABLES

Set the run mode for the L<Mojolicious> application to C<development>.

    PNI_DEV=1

=cut
